# Bug Invaders
Arcade style game with some modern twists. Bug Invaders uses vanilla javascript to rewrite css positioning of static images in
order to create animations within the game. 

FRAMEWORKS:
The backend is built using a Ruby on Rails framework while the frontend is built on a Javascript/HTML/CSS.
FEATURES:
- Play the game bug invaders.
- Set personal high scores
- Set all-time high scores
- Leave comments
HOW TO USE:
Create an account by signing up with relavent user information. Press start game button below television to begin game prompt.
One can change their user avatar by clicking the change avatar button on screen.
